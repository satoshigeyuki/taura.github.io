Research

My main theme is parallel processing and programming languages. Recent topics are substrate software for Global computing, languages for shared-memory parallel programming, and parallel and distributed memory management (GC).

  • Phoenix : Grid programming and tools project
  • StackThreads/MP : super light-weight threads project
  • MTCAMP : parallel programming language project
  • Parallel and distributed GC project
